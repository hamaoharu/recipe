# 2026-05-27

## 1) 今日わかったこと
- Props と State の役割を整理できた。Props は親から渡す値、State はコンポーネント自身が持つ可変データ。
- `onClick={handleX}` と `onClick={() => handleX(arg)}` の違いを理解した。引数を渡したいときはアロー関数で包む。
- 条件分岐は `? :` と `&&` の使い分けで読む。表示切替は `? :`、条件付き表示は `&&` が基本。

## 2) 詰まったこと / 質問
- JSX の括弧（`()`, `{}`, `[]`）の役割が混ざりやすかった。特に `children` と分割代入で混乱した。
- `className` の組み立てはテンプレートリテラルと配列 `join(" ")` の両方があるので、どちらを使うかで迷った。
- Render Props の定義が曖昧だったが、「描画用の関数を props として渡す」ものだと整理した。
