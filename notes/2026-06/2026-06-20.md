- try catch 失敗したらcatchの処理
```
try {
  // ここで失敗するかもしれない処理
} catch {
  // 失敗したらここへ来る
}
```

- オブジェクトの定義と取得を同時に行なっている obj[tab]と同じ
- 一時的にオブジェクトを利用しているだけで、中身は配列になる
```
const tabRoadmaps = {
    posts:     myRoadmaps,
    likes:     likedRoadmaps,
    bookmarks: bookmarkedRoadmaps,
}[tab];
```

- ジェネリクスは初期値よりも広い型をTSに伝えるための書き方 nullだと狭すぎる
- | nullを書くのは初期値がnullの時だけ
```
const [user, setUser]       = useState<Author | null>(null);
```

- エラーの種類

| 種類 | 対処 |
| --- | --- |
| 推論が狭い | ジェネリクスで型を広げる |
| 推論が広い | asで型を狭める |
| 取り出しが不安 | キーの型を限定 |
| 外部データ | asで型を教える |
